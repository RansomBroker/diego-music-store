# Folder Structure Documentation

This document describes the folder layout of the ERP project and the tracking system `.workspace_ai/`.

## ERP Project Directory
- `KEBUTUHAN_APLIKASI_BISNIS.md` - Raw business specifications.
- `PRD_DIEGO_MUSIC_STORE.md` - Full Product Requirements Document.
- `timeline_pengerjaan.md` - Timeline and sprint distribution charts.

## Workspace AI metadata (`.workspace_ai/`)
- `workspace/` - Project specifications, epics, features, architecture decisions.
- `execution/` - Project task tracking, Sprints backlog, progress reports.
- `architecture/` - Standard architectures, diagrams, and coding standards.
- `github/` - Mappings to GitHub Project Board, Issues, and Milestones.
- `roles/` - Product Manager, Architect, Developer, Reviewer, Doc Writer responsibilities.
- `templates/` - Reusable templates for features, tasks, and reviews.
- `logs/` - Logs of workflow sessions, sync runs, activity logs.
- `settings/` - Workflow stages, GitHub repository links, project metadata.
