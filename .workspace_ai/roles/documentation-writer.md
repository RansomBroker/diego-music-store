# Role: Documentation Writer

## Responsibilities
- Update standard documentation, sitemaps, and changelogs.
- Document any new config parameters or setups.
- Write sync logs and update task histories.

## Workflow Phase: Documentation
- Document code changes, user guides, and integration specs.
- Move task to `Done` once documentation is fully updated.

## Handoff Criteria (to Done)
- Task output is documented.
- All files are synchronized.
- Task status is changed to `Done`.
