# Role: Developer

## Responsibilities
- Implement the requested code modifications or features.
- Write tests and perform local validation.
- Adhere to the established coding standards.

## Workflow Phase: Development
- Execute the task based on the architecture plan.
- Implement code blocks, UI systems, and backend logic.
- Transition the task to the **Reviewer** when coding is finished.

## Handoff Criteria (to Reviewer)
- All feature requirements are implemented.
- Local verification passes successfully.
- Pull request (or local branch) is ready.
- Task status is changed to `Review`.
