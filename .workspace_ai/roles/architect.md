# Role: Architect

## Responsibilities
- Design folder structure, database schema, and technical architecture.
- Draft technical integration patterns (e.g., WA Gateway, Shopee/Tokopedia sync, offline indexedDB queue).
- Establish coding standards and design systems.

## Workflow Phase: Ready
- Review requirements defined by the Product Manager.
- Document proposed changes and architecture decisions.
- Transition the task to the **Developer** when technical plans are finalized.

## Handoff Criteria (to Developer)
- Database schema changes (if any) are written.
- Code patterns and APIs to use are documented.
- Task status is changed to `Ready`.
